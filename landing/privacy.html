Política de Privacidade

Última actualização: 30 de Maio de 2026

Este é um modelo inicial. Recomendamos a revisão por um profissional jurídico antes da utilização comercial definitiva.

1. Quem somos

O Klaru é uma plataforma de CRM. Esta política explica como tratamos os dados pessoais de quem usa o Serviço.

2. Dados que recolhemos

Dados de conta (nome, email, telefone), dados da empresa, e os dados que o utilizador introduz no CRM (contactos, mensagens, negócios). Recolhemos também dados técnicos de utilização para segurança e melhoria do Serviço.

3. Como usamos os dados

Para fornecer e manter o Serviço, processar pagamentos, dar apoio, garantir segurança e cumprir obrigações legais. Não vendemos dados pessoais.

4. Integrações de terceiros

O Klaru liga-se a serviços como o WhatsApp (Meta), Instagram, Facebook, email e fornecedores de IA. Ao usar essas integrações, os dados necessários são partilhados com esses fornecedores segundo as políticas deles.

5. Segurança

Aplicamos medidas técnicas e organizativas para proteger os dados, incluindo encriptação de credenciais sensíveis e controlo de acessos. Nenhum sistema é 100% infalível, mas comprometemo-nos com boas práticas.

6. Retenção

Mantemos os dados enquanto a conta estiver activa e pelo período necessário ao cumprimento de obrigações legais. Após o encerramento, os dados são eliminados ou anonimizados num prazo razoável.

7. Os seus direitos

O utilizador pode aceder, corrigir, exportar ou pedir a eliminação dos seus dados pessoais, contactando-nos. Responderemos dentro dos prazos aplicáveis.

8. Responsabilidade do cliente

Quando o utilizador carrega dados dos seus próprios contactos, age como responsável pelo tratamento desses dados e deve assegurar uma base legal adequada (por exemplo, consentimento).

9. Alterações

Podemos actualizar esta política. As alterações relevantes serão comunicadas pelos meios habituais.

10. Contacto

Para questões de privacidade ou exercício de direitos: qdcreat@gmail.com.
